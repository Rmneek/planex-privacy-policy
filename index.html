Privacy Policy

Plan Execute Next respects your privacy. This Privacy Policy explains how we collect, use, and protect your information when you use our mobile application.

1. Information We Collect

  • Email address and basic profile information (via Google Sign-In)
  • User identifiers for authentication
  • Task data and reminder preferences entered by the user
  • App usage data for performance and reliability

2. How We Use Information

We use collected information to:

  • Provide authentication and app functionality
  • Deliver reminders and daily summary notifications
  • Improve app stability and user experience

3. Notifications & Alarms

The app sends notifications only with the user’s consent. These include daily summaries and task reminders.

  • POST_NOTIFICATIONS – to show reminders and summaries
  • RECEIVE_BOOT_COMPLETED – to restore reminders after device restart
  • SCHEDULE_EXACT_ALARM – to deliver notifications at user-selected times (e.g., daily summary at 10:00 AM)

4. Third-Party Services

We use the following third-party services:

  • Firebase Authentication
  • Firebase Firestore
  • Firebase Cloud Messaging
  • Google Sign-In

5. Data Security

We implement industry-standard security measures to protect your data.

6. User Control

You can disable notifications or request data deletion at any time.

8. Changes

This policy may be updated from time to time.
